### [Watchlist API](https://github.com/anass-hajjaji/watchlist-api) — REST API with Full Auth
> `Node.js` · `Express.js` · `PostgreSQL` · `Prisma` · `JWT` · `Zod`

Production-ready REST API for managing movies and personal watchlists.

- JWT stored in secure `httpOnly` cookies, bcrypt hashing, ownership-based access control
- Zod validation on every request body + UUID checks to avoid unnecessary DB hits
- Relational schema with a `userId + movieId` unique constraint to prevent duplicate watchlist entries
